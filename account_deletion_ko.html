방법 1. 앱 내에서 직접 삭제 (권장)

앱을 여전히 사용 중이고 Google 로그인 상태인 경우, 다음 절차로 즉시 계정과 모든 클라우드 데이터를 영구 삭제할 수 있습니다.

  1. 앱 실행 → 화면 우측 상단 메뉴(☰) 버튼
  2. "내 계정" 선택
  3. 화면 하단의 "계정 삭제" 버튼
  4. 경고 다이얼로그에서 "영구 삭제" 확인

처리는 즉시 이루어지며, 별도 대기 시간이나 추가 확인 절차 없이 데이터가 영구 삭제됩니다.
